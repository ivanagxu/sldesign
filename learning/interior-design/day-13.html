** 餐厅空间规划与设计要点

📖 Day 13: 餐厅设计详解

阶段： 进阶 → Week 2 → Day 13
主题： 餐厅空间规划与设计要点
用时： 60 分钟
日期： 2026-04-08

🎯 学习目标

学完后你能： - ✅ 掌握餐厅空间的最小尺寸要求 - ✅ 设计适合小户型的餐厅布局 - ✅ 选择合适的餐桌椅组合 - ✅ 规划餐厅照明和收纳系统

一、知识讲解（15 分钟）

1.1 餐厅设计的核心问题

在小户型（如 37㎡）中，餐厅往往是最容易被牺牲的空间。但良好的餐厅设计能： - 提升用餐体验和仪式感 - 增加家庭互动机会 - 作为多功能空间使用（工作/社交）

常见误区： - ❌ 为了省空间取消餐厅 - ❌ 餐桌尺寸过大导致动线拥挤 - ❌ 忽视照明设计影响用餐氛围 - ❌ 没有规划餐具收纳空间

1.2 餐厅空间尺寸标准

最小用餐空间要求

人数 餐桌尺寸 (长×宽) 所需空间 (含椅子) 适用场景
1-2 人 60×60cm 1.2×1.2m 单身/情侣
2-4 人 120×70cm 2.0×1.5m 小家庭
4-6 人 160×80cm 2.5×2.0m 大家庭

椅子活动空间

┌─────────────────────────────────────────┐
│                                         │
│          餐桌                           │
│     ┌───────────────┐                   │
│     │               │                   │
│     │               │                   │
│     └───────────────┘                   │
│         ↑                               │
│    至少 75cm (椅子拉出 + 人通过)          │
│                                         │
└─────────────────────────────────────────┘

关键尺寸： - 椅子拉出需要：45cm - 人通行需要：30cm - 总计最小预留：75cm（从桌边到墙/障碍物）

1.3 小户型餐厅布局方案

方案 A：独立餐厅（适合 37㎡）

┌─────────────────────────────────────┐
│                                     │
│   ┌───────────┐                     │
│   │   餐桌    │    过道 75cm+       │
│   │ 120×70cm  │                     │
│   └───────────┘                     │
│                                     │
│   靠墙卡座 (可选)                    │
└─────────────────────────────────────┘

优点： 独立用餐空间，仪式感强
缺点： 占用面积较大

方案 B：客餐一体（推荐 37㎡）

┌─────────────────────────────────────┐
│                                     │
│    沙发区        餐桌区              │
│   ┌───────┐    ┌──────┐             │
│   │ L 型   │    │ 餐桌 │             │
│   │ 沙发  │    │      │             │
│   └───────┘    └──────┘             │
│                                     │
│         共用活动空间                 │
└─────────────────────────────────────┘

优点： 空间通透，视觉更大
缺点： 功能分区不明显

设计要点： - 用灯具/地毯划分区域 - 餐桌与沙发距离≥1m - 风格统一保持视觉连贯

方案 C：吧台替代（超小户型）

┌─────────────────────────────────────┐
│                                     │
│   ┌───────────────────────────┐     │
│   │       吧台 (60cm 深)       │     │
│   │   ○    ○    ○    ○       │     │
│   │  吧椅  吧椅  吧椅  吧椅    │     │
│   └───────────────────────────┘     │
│                                     │
│          厨房/客厅                   │
└─────────────────────────────────────┘

优点： 最省空间，可兼作工作台
缺点： 用餐舒适度较低

适用场景： - 单身/情侣 - 很少在家正式用餐 - 需要额外工作台

1.4 餐桌椅选择指南

餐桌形状对比

形状 优点 缺点 适用场景
长方形 利用率高，靠墙摆放灵活 角落空间难用 大多数户型
正方形 适合小空间，四人均等 扩展性差 2 人用餐
圆形 无棱角安全，氛围好 占空间，难靠墙 有小孩家庭
椭圆形 兼具长方和圆形优点 选择较少 追求品质

椅子类型选择

类型 宽度 特点 建议
餐椅 45-50cm 标准舒适 首选
长凳 可定制 省空间，可收纳 一侧靠墙
卡座 定制 收纳 + 座椅 靠墙最优
吧椅 35-40cm 高脚，省地 吧台配套

计算公式：

每人用餐宽度 = 60cm (舒适) / 50cm (紧凑)
餐桌长度 = 人数 × 每人宽度

二、37㎡户型餐厅设计方案（20 分钟）

2.1 原始户型分析

假设 37㎡户型为典型一室一厅： - 客厅开间：3.2m - 餐厅位置：入户旁/客厅一侧 - 可用餐厅空间：约 2.5×2.0m

2.2 推荐方案：客餐一体 + 卡座设计

┌─────────────────────────────────────────────┐
│  入户      卡座 (带收纳)                     │
│   ↓      ┌──────────────────┐               │
│         │    软包靠背       │               │
│  鞋柜   ├──────────────────┤               │
│         │    餐桌 120×70    │  过道          │
│         │                   │  80cm         │
│         └──────────────────┘               │
│                ↑                           │
│              餐椅×2                         │
│                                            │
│              客厅区                         │
└─────────────────────────────────────────────┘

设计亮点： 1. 卡座替代 2 把椅子：节省约 40cm 进深 2. 卡座下方收纳：存放餐具/杂物 3. 长凳可移动：灵活调整座位数 4. 视觉通透：不遮挡客厅光线

2.3 材料建议

项目 推荐材料 预算 理由
餐桌 实木/岩板 800-2000 元 耐用易清洁
卡座 定制板材 + 软包 1500-3000 元 舒适 + 收纳
餐椅 实木/塑料 200-500 元/把 轻便易搬动
地面 与客厅通铺 - 视觉统一

三、餐厅照明设计（10 分钟）

3.1 照明层次

┌─────────────────────────────────────────┐
│                                         │
│         基础照明 (筒灯/吸顶灯)           │
│              ↓                          │
│         重点照明 (吊灯/餐吊灯)           │
│              ↓                          │
│         氛围照明 (灯带/蜡烛)             │
│                                         │
└─────────────────────────────────────────┘

3.2 餐吊灯安装标准

参数 标准值 说明
离桌面高度 70-80cm 不遮挡视线
灯具直径 餐桌宽度的 1/2-2/3 比例协调
色温 2700-3000K 暖光增进食欲
照度 150-200 lux 桌面足够亮

3.3 推荐配置

基础配置： - 1 盏餐吊灯（餐桌正上方） - 2-3 个筒灯（周围补光）

进阶配置： - 1 盏主吊灯 + 2 盏小吊灯（长餐桌） - 卡座上方灯带（氛围） - 可调光开关（场景切换）

四、餐厅收纳系统（10 分钟）

4.1 收纳物品清单

类别 物品 建议位置
餐具 碗筷盘勺 餐桌下方/餐边柜
杯具 水杯/酒杯 吊柜/开放格
食品 茶叶/咖啡/零食 餐边柜/抽屉
小电器 热水壶/咖啡机 餐边柜台面
杂物 餐巾纸/牙签 桌面收纳盒

4.2 小户型收纳方案

方案 A：卡座下方收纳

┌──────────────────────┐
│     软包坐垫        │ ← 可掀开
├──────────────────────┤
│  ┌────┐ ┌────┐      │
│  │    │ │    │      │ ← 分隔收纳
│  │    │ │    │      │
│  └────┘ └────┘      │
└──────────────────────┘

收纳容量： 约 200L（相当于 2 个大行李箱）

方案 B：墙面薄柜

  • 深度：20-25cm（不占过道）
  • 高度：到顶（最大化利用）
  • 功能：上柜放杯具，下柜放餐具

方案 C：移动推车

  • 尺寸：30×40cm
  • 层数：3 层
  • 用途：临时收纳/上菜车
