** 玄关与过道设计

🏠 室内设计培训课程

Day 12：玄关与过道设计

日期： 2026-04-07
周次： Week 2 - Day 5
课题： 玄关与过道设计
预计学习时间： 45-60 分钟

📌 学习目标

  1. 理解玄关的功能定位与设计原则
  2. 掌握过道空间的优化技巧
  3. 学会在小户型中高效利用过渡空间
  4. 能够针对 37㎡户型设计合理的玄关与过道方案

一、玄关设计核心原则

1.1 玄关的三大功能

功能 说明 设计要点
缓冲过渡 室内外空间的心理过渡区 视觉遮挡，避免"开门见山"
收纳整理 鞋、外套、钥匙、快递等 分类收纳，就近原则
形象展示 家的第一印象 灯光、装饰、整洁度

1.2 玄关类型

独立玄关 - 有独立空间，可设计完整鞋柜 + 换鞋凳 + 挂衣区 - 37㎡户型较难实现

入户花园式玄关 - 利用入户门旁小空间 - 适合狭长户型

开门见厅式玄关 - 最常见于小户型 - 需要巧妙设计隔断/屏风 - 37㎡户型首选方案

1.3 玄关尺寸参考

最小舒适尺寸：
- 宽度：≥ 90cm（单人通行）
- 深度：≥ 35cm（标准鞋柜深度）
- 高度：通顶设计（最大化收纳）

鞋柜内部尺寸：
- 层板高度：15-20cm（平底鞋）、20-25cm（高跟鞋/靴子）
- 活动层板：可调节，适应不同鞋型
- 底部留空：15-20cm（常穿鞋免开门收纳）

二、玄关收纳系统设计

2.1 鞋柜设计要点

分区设计：

┌─────────────────────────┐
│      顶部：换季鞋盒      │  ← 不常用物品
├─────────────────────────┤
│      中部：挂衣区        │  ← 外套、包包
│      + 置物台            │  ← 钥匙、快递
├─────────────────────────┤
│      下部：日常鞋柜      │  ← 当季鞋子
├─────────────────────────┤
│      底部：常穿鞋位      │  ← 免开门，直接踢入
└─────────────────────────┘

关键细节： - 中部留空高度：40-50cm（方便放包、小物件） - 挂衣区：设置挂钩或伸缩挂衣杆 - 底部留空：15-20cm，嵌入感应灯带 - 柜门：推荐无拉手设计（按压式/凹槽），视觉简洁

2.2 玄关附加功能

功能 实现方式 适用场景
换鞋凳 嵌入式/翻折式/独立式 有老人小孩家庭必备
穿衣镜 柜门镜/墙面镜/独立镜 出门前整理仪容
雨伞收纳 底部侧边预留空间 多雨地区
快递暂存 中部置物台延伸 网购频繁家庭
消毒区 小型紫外线消毒盒 疫情后时代需求

三、过道设计优化

3.1 过道常见问题

问题 1：狭长压抑 - 原因：宽度不足、灯光昏暗、墙面单调 - 解决：增加照明、浅色墙面、镜面延伸

问题 2：浪费空间 - 原因：仅作为通道，无功能整合 - 解决：墙面收纳、嵌入式柜体

问题 3：动线冲突 - 原因：开门方向、家具摆放不当 - 解决：调整开门方向、优化家具布局

3.2 过道宽度标准

类型 最小宽度 舒适宽度
单人通行 60cm 80cm
双人错身 90cm 120cm
搬运家具 - ≥ 100cm

3.3 过道收纳技巧

墙面利用： - 薄柜：深度 15-20cm，收纳书籍、装饰品 - 挂钩区：挂画、外套、包包 - 壁龛：非承重墙可挖壁龛，深度 10-15cm

端景设计： - 过道尽头墙面是视觉焦点 - 挂画、绿植、小型雕塑 - 灯光重点照射，提升空间品质

四、37㎡户型玄关与过道实战

4.1 典型 37㎡户型玄关类型

类型 A：入户即客厅（最常见）

┌──────────────────┐
│                  │
│   玄关柜 ←┌门    │
│           │      │
│   ┌───────┘      │
│   │   客厅       │
│   │              │
└───┴──────────────┘

设计策略： - 利用入户门一侧墙面做通顶鞋柜 - 柜体侧面可做长虹玻璃隔断，避免开门见厅 - 底部留空 + 感应灯，视觉轻盈

类型 B：入户有短过道

┌──────────────────┐
│  鞋柜 │  过道    │
│  ←───→│  ←───→  │
│  35cm │  90cm   │
│       │   门    │
│       │   ↓     │
│       └─────────│
│         客厅    │
└─────────────────┘

设计策略： - 单侧鞋柜 + 墙面挂钩 - 过道尽头设置端景 - 灯光分层：基础照明 + 重点照明

4.2 37㎡户型玄关设计禁忌

❌ 忌：鞋柜过深 - 超过 40cm 会占用过多空间 - 标准深度 35cm 足够

❌ 忌：全封闭柜门 - 小空间更显压抑 - 中部/底部适当留空

❌ 忌：复杂造型 - 简约为主，避免视觉杂乱 - 颜色与整体空间协调

❌ 忌：忽视灯光 - 玄关是回家第一盏灯 - 建议：主灯 + 柜内灯 + 底部感应灯

4.3 推荐方案配置

经济型（预算 3000-5000 元） - 成品鞋柜 + 墙面挂钩 + 穿衣镜 - 优点：灵活、可移动 - 缺点：收纳量有限

定制型（预算 8000-15000 元） - 通顶定制鞋柜 + 换鞋凳 + 挂衣区 - 优点：最大化收纳、整体美观 - 缺点：不可移动、工期长

智能型（预算 15000+ 元） - 定制柜体 + 智能感应 + 消毒杀菌 + 恒温恒湿 - 优点：功能齐全、体验好 - 缺点：成本高、维护复杂

五、灯光设计要点

5.1 玄关灯光层次

层次 灯具类型 作用
基础照明 筒灯/吸顶灯 整体照亮空间
功能照明 柜内灯/镜前灯 方便取物、整理
氛围照明 灯带/壁灯 提升空间质感

5.2 色温选择

  • 推荐：3000K-3500K（暖白光）
  • 温馨舒适，适合回家放松氛围
  • 避免冷白光（>4000K），显得生硬

5.3 智能感应

  • 人体感应灯： 进门自动亮，离开自动灭
  • 柜内感应灯： 开门即亮，方便取物
  • 底部感应灯带： 夜间回家不刺眼

六、课后作业

6.1 思考题

  1. 测量你家入户区域的尺寸（宽度、深度、高度）
  2. 列出你目前的玄关痛点（如：鞋子没地方放、找不到钥匙等）
  3. 根据今天所学，画出你理想中的玄关布局草图

6.2 实践题

针对 37㎡户型设计一个玄关方案：

要求： - 包含鞋柜（至少容纳 20 双鞋） - 有挂衣区 - 有置物台（放钥匙、快递） - 考虑灯光设计 - 标注主要尺寸
