一、小户型卫生间设计核心原则

1.1 空间最大化原则

策略 说明 适用场景
干湿分离 淋浴区独立，保持干区干燥 面积≥3㎡
一体化设计 洗漱+淋浴+马桶整合 面积<3㎡
壁挂式洁具 减少地面占用，便于清洁 所有小户型
镜柜收纳 垂直空间利用 所有户型

1.2 37㎡户型卫生间参考尺寸

标准小卫生间：1.5m × 1.8m = 2.7㎡
紧凑卫生间：1.2m × 1.5m = 1.8㎡
最小可用尺寸：0.9m × 1.2m = 1.08㎡ (仅马桶 + 小洗手台)

1.3 人体工学关键尺寸

项目 最小尺寸 舒适尺寸
马桶前方活动空间 45cm 60cm
淋浴区最小尺寸 80×80cm 90×90cm
洗手台宽度 50cm 60-80cm
洗手台高度 80-85cm 85-90cm
通道宽度 60cm 75cm+

二、卫生间布局方案

2.1 常见布局类型

方案 A：一字型布局 (适合狭长空间)

┌─────────────────────────┐
│ 洗手台  │  马桶  │ 淋浴区 │
│  60cm   │  70cm  │  90cm  │
└─────────────────────────┘
        总长度约 220cm

优点： 动线流畅，管道集中
缺点： 需要较长墙面

方案 B：L 型布局 (适合方形空间)

┌──────────────┬──────────┐
│   淋浴区     │          │
│   90×90cm    │   马桶   │
│              │          │
├──────────────┤          │
│   洗手台     │          │
│   60cm       │          │
└──────────────┴──────────┘

优点： 空间利用率高，干湿分离
缺点： 管道布置较复杂

方案 C：钻石型淋浴房 (最小空间方案)

┌─────────────────────────┐
│  洗手台  │    /         │
│          │   / 淋浴区   │
│          │  /  钻石型   │
│   马桶   │ /           │
└──────────┴─────────────┘

优点： 节省空间，美观
缺点： 淋浴空间略小

三、关键工艺要点

3.1 防水工程 (重中之重!)

区域 防水高度 注意事项
淋浴区 ≥180cm 满墙防水最佳
洗手台区域 ≥120cm 超出台面 30cm
其他墙面 ≥30cm 地面上翻
地面 全地面 + 上翻 30cm 门口做止水坎

防水施工流程： 1. 基层处理 (清理、找平、润湿) 2. 节点处理 (管根、墙角做圆弧) 3. 第一遍防水 (纵向涂刷) 4. 第二遍防水 (横向涂刷，间隔 4-8 小时) 5. 闭水试验 (48 小时，到楼下检查)

3.2 排水设计

常见排水方式：

类型 优点 缺点 适用
传统地漏 成本低，易维修 易堵塞，有异味 预算有限
长条形地漏 排水快，美观 成本高，安装要求高 推荐
隐藏式地漏 美观一体 维修困难 高端设计
墙排式 无地面地漏，易清洁 成本高，需预埋 极简风格

排水坡度要求： - 标准坡度：1%-3% (每米下降 1-3cm) - 地漏位置：最低点，远离门口

3.3 通风与防潮

通风方案优先级： 1. ✅ 外窗自然通风 + 排风扇 (最佳) 2. ✅ 排风扇 + 止逆阀 (无窗必备) 3. ⚠️ 新风系统辅助 (成本高)

防潮措施： - 使用防水石膏板或铝扣板吊顶 - 墙面使用瓷砖或防水涂料 - 门套底部做防潮处理 (石材或金属) - 安装风暖浴霸 (兼顾取暖 + 干燥)

四、收纳系统设计

4.1 垂直收纳策略

墙面收纳系统 (从下到上)
┌─────────────────────┐
│    吊柜/镜柜        │ ← 备用物品、清洁剂
│    (高度 60-80cm)    │
├─────────────────────┤
│    毛巾架/置物架    │ ← 日常用品
│    (高度 120-150cm)  │
├─────────────────────┤
│    洗手台台面       │ ← 洗漱用品
│    (高度 85cm)       │
├─────────────────────┤
│    台盆柜           │ ← 囤货、大件物品
│    (高度 60-70cm)    │
└─────────────────────┘

4.2 镜柜设计要点

参数 推荐值 说明
镜柜深度 12-15cm 太深显得压抑
镜柜高度 60-80cm 根据层高调整
开放格比例 20-30% 放置常用物品
镜前灯 必备 色温 4000K 自然光

4.3 淋浴区收纳

  • 壁龛： 装修时预埋，最美观 (推荐)
  • 三角篮： 免打孔，灵活调整
  • 淋浴杆置物： 多功能整合

五、材料选择建议

5.1 瓷砖选择

区域 推荐类型 规格 注意事项
地面 防滑砖 300×300mm 防滑系数 R10+
墙面 釉面砖 300×600mm 易清洁
淋浴区 马赛克/小砖 100×100mm 便于找坡
美缝 环氧彩砂 - 防水防霉

5.2 洁具选择

马桶： - 小户型推荐：壁挂式 (节省空间，易清洁) - 坑距确认：305mm 或 400mm - 预算参考：2000-5000 元 (智能盖板另计)

花洒： - 恒温花洒：提升舒适度 (推荐) - 空气注入：节水且舒适 - 预算参考：1500-4000 元

浴室柜： - 材质：多层实木/铝合金 (防水) - 台盆：陶瓷一体盆 (易清洁) - 预算参考：2000-6000 元

六、37㎡户型卫生间设计任务

6.1 设计条件

假设你的 37㎡户型卫生间尺寸：1.5m × 1.8m = 2.7㎡

6.2 设计要求

  1. 必须实现干湿分离
  2. 需要容纳： 马桶、洗手台、淋浴区、收纳空间
  3. 风格： 现代简约
  4. 预算： 3-5 万元 (含硬装 + 洁具)

6.3 设计输出

请绘制/描述以下内容：

□ 平面布局图 (标注尺寸)
□ 立面图 (至少一面墙)
□ 材料清单 (瓷砖、洁具、五金)
□ 收纳系统说明
□ 灯光设计 (主灯 + 镜前灯 + 氛围灯)

6.4 参考案例

2.7㎡卫生间成功案例要素： - ✅ 钻石型淋浴房 (90×90cm) - ✅ 壁挂马桶 + 上方吊柜 - ✅ 镜柜延伸至马桶上方 - ✅ 长条形地漏 + 墙排下水 - ✅ 浅色系瓷砖 (视觉放大)

七、常见避坑指南

坑点 后果 预防措施
防水没做好 楼下漏水，返工成本高 闭水试验 48 小时，必须检查
地漏位置不当 积水、排水慢 贴砖前确认最低点
插座没留够 智能马桶/电热毛巾架无法安装 马桶旁预留 1-2 个插座
排风扇未装止逆阀 异味倒灌 必须安装止逆阀
美缝剂选错 发霉变黑 选环氧彩砂，不要用普通填缝剂
门套未防潮 门套底部发黑腐烂 门套底部用石材/金属
收纳不足 后期杂乱 设计阶段预留充足收纳
